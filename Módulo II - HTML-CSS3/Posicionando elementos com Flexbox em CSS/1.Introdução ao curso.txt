Introdução ao Flex Box

	O Flex Box foi projetado como modelo de layout unidimensional e como método que pode oferecer distribuição de espaço entre itens em uma interface e recursos de alinhamento.

Flex Container: É a tag que envolve os itens, será nela que iremos aplicar a propriedade “display: flex”. Transforma todos os seus filhos em flex itens.
	Propriedades:
*display		*justify-content
*flex0direction		*align-items
*fles-wrap		*align-content
*flex-flow

Flex item: São elementos filhos diretos do Flex Container. E também podem se tornar Flex Container
	Propriedades:
*flex-grow		*order
*flex-basis		*align-self
*flex-shrink
*flex
