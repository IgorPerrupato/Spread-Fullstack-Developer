Aula 4: This


	A palavra this é uma referência de contexto.

Seu valor pode mudar de acordo com o lugar no código onde foi chamada.

Contexto			Referencia
Em um objeto (método)	Próprio objeto dono de método
Sozinha			Objeto global (em navegadores, window)
Função			Objeto global
Evento			Elemento que recebeu o evento


Manipulando seu valor


Call
	Se refere ao objeto que foi informado na chamada call. É possível passar parâmetros para essa função separando-os por virgulas. 


Apply
	É possível passar parâmetros para essa função dentro de um array.


Bind
	Clona a estrutura da função onde é chamada e aplica o valor do objeto passado como parâmetro.
